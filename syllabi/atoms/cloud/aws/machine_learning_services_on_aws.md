# Machine learning services on `AWS`

## Credits
Mark Veltzer [mark.veltzer@gmail.com](mailto:mark.veltzer@gmail.com)

## Description
This course is intended for machine learning engineers who wish to get acquainted with
the services `AWS` offers with regard to machine learning. The course does not teach
machine learning, only how to use the tools `AWS` offers to achieve the goal.

Tools which are discussed are infrastructure dealing, machine learning and deep
learning as well as more advanced ready tools such as chatbot tools, transcription and
translation, and `NLP` tools such as comprehend.

This course does not provide extensive training on these tools as the offering is huge
and it will take weeks to go through all of these in detail. Instead it provides an
overview of the services with examples so that the machine learning engineers gets an
idea of what's on offer and will save himself/herself time by using these services.
Further learning of individual services is required.

## Duration
16 hours / 2 days

## Intended Audience
* Infrastructure people who need to set up infrastructure for machine learning at
scale on `AWS`.
* Machine learning engineers who would like to use `AWS` services to do machine learning.
* Managers who manage machine learning engineers who would like to understand what's on
off in `AWS` in the field of machine learning.

## Prerequisites
Knowledge of machine learning.

## Objectives
* List the various services that are on offer in `AWS` regarding machine learning.
* Understand the advantages and disadvantages of these services when compared to each other.
* Understand the advantages and disadvantages of these services when compared to doing the same off cloud.
* Understand the advantages and disadvantages of these services when compared to doing the same on cloud but without the service.

## Exercises
No exercises in this part of the course, examples for some of the services will be
shown by the instructor.

## Outline
* Classic `ML`
    * Amazon SageMaker
    * Amazon SageMaker Neo
* Labeling services
    * Amazon SageMaker Ground Truth
    * Amazon Mechanical Turk
* Infrastructure for machine learning
    * Infrastructure for `Spark` in `AWS` via `EMR`
    * Infrastructure for storage via `S3`
    * `S3` as data lake
* Machine infrastructure
    * Amazon Elastic Inference
    * `AWS` Deep Learning AMIs
* `NLP` services
    * Amazon comprehend
    * `AWS` Lex
* Text and speech
    * `AWS` Polly
    * Amazon Transcribe
    * Amazon Translate
* Time Series prediction
    * `AWS` Forecast
* Personalized recommendations
    * `AWS` Personalize
* Image and video
    * Amazon Rekognition
* Image to text
    * Amazon Textract
