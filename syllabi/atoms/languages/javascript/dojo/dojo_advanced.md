# Dojo Advanced

## Credits
Mark Veltzer [mark.veltzer@gmail.com](mailto:mark.veltzer@gmail.com)

## Description
This course covers advanced `Dojo` toolkit topics including custom widget development, data handling with stores, application architecture, internationalization, and performance optimization. Students will gain hands-on experience building complex widget-based web applications.

## Duration
24 hours / 3 days

## Intended Audience
Web developers with basic `Dojo` experience who need to build advanced widget-based applications.

## Prerequisites
* Working knowledge of `JavaScript` and `HTML`/`CSS`
* Basic experience with the `Dojo` toolkit

## Outline
* classes in dojo, multi inheritance
    * dojo - quick overview on different widgets
    * widgets - life cycle
    * custom widget - how to write
    * event handling in widgets and in inner widgets
    * custom widget - exercise
* data handling, dojo.data, dojo.io
    * data stores- store types and when to use
    * `JSON` input to store, objects etc...
    * trees
    * tables
    * data and store - exercise
* advanced topics
    * layouts - overview
    * require, on
    * wire
    * i18n in dojo
    * data binding
    * application architecture on dojo
        * application level
        * module level
        * page level
    * memory and garbage collection
    * data structures on dojo
    * tools and debugging
    * build and deployment
    * i18n in general and in widget
