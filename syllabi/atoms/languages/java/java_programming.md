# `Java` Programming

## Credits
Mark Veltzer [mark.veltzer@gmail.com](mailto:mark.veltzer@gmail.com)

## Description
A full and comprehensive course about the `Java` programming language and environment. The course is intended for
beginning developers and not seasoned developers. The course explains the basic workings of the `JVM`, how to
write code in `Java`, how to structure your code and how to use the basic libraries. It also teaches the basics
of multi-threading in `Java`.

## Duration
40 hours / 5 days

## Intended Audience
* People with no programming experience.
* Developers who wish to study the basics of the `Java` programming language.

## Prerequisites
* Tech affinity.
* Any experience in any other programming language is an advantage.

## Objectives
* Write, test and debug `Java` applications.
* Use the `Java` standard libraries.
* Use the basic `Java` data structures supplied with the standard library.
* Write multi-threaded `Java` applications.
* Work with development environments for the `Java` programming language.
