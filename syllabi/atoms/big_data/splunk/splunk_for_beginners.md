# `Splunk` for Beginners

## Credits
Mark Veltzer [mark.veltzer@gmail.com](mailto:mark.veltzer@gmail.com)

## Description
This beginner-friendly course introduces `Splunk`, a powerful platform for operational intelligence and log management. Students will learn how to navigate the `Splunk` interface, run basic searches, understand search results, and create simple reports. No prior `Splunk` experience required.

## Duration
8 hours / 1 day

## Intended Audience
* IT professionals new to log management
* Developers wanting to learn `Splunk` basics
* System administrators with no prior `Splunk` experience

## Prerequisites
* Basic understanding of IT concepts
* Familiarity with log files

## Objectives
* Navigate the `Splunk` interface.
* Run basic searches and understand search results.
* Use basic stats commands.
* Create simple reports and charts.

## Outline
* Getting Started with `Splunk`
    * What is `Splunk` and why use it
    * `Splunk` architecture overview (indexers, search heads, forwarders)
    * Introduce `Splunk` and the Search app
    * Navigating the `Splunk` interface
* Running Your First Searches
    * Run basic searches
    * Time range selection and time-based searching
    * Identify the contents of search results
    * Control a search job
    * Use the output of a search to refine your search
* Search Fundamentals
    * Examine the anatomy of a search
    * Understand search language syntax concepts
    * Review fields and use the fields command
    * Create a table
* Basic Statistics
    * Understand the stats command
    * Display top and rare values for given fields
* Creating Simple Reports
    * Create reports and charts
    * Chart functions (timechart)
    * Introduction to dashboards
