# Wi-Fi Protocols

## Credits
Mark Veltzer [mark.veltzer@gmail.com](mailto:mark.veltzer@gmail.com)

## Description
WiFi communication is probably the most common communication used by end users today. This course
introduces engineers to wireless networking protocols and centers mostly around 802.11 and
specifically 802.11b.

## Duration
24 hours / 3 days

## Intended Audience
* Software engineers who want to understand the inner workings of WiFi networks.
* Low level or Operating system software engineers who want to implement some of these protocols

## Prerequisites
* This course requires some kind of engineering degree to be intelligible.
* The protocol section requires some understanding of what computer protocols are.
* Some math is required for understanding the error correction and multiplexing ideas.

## Objectives
* To get a grasp of the underlying protocols underlying current WiFi technology.
* To understand the basics of security under these protocols.

## Outline
* Underlying ideas
    * OFDM
    * MIMO
    * BeamForming
    * SpacialMultiplexing
* Protocols review
    * IEEE 802.11 Wi-Fi protocol summary
        * 802.11ax (Wi-Fi 6)
        * 802.11ac wave2
        * 802.11ac wave1
        * 802.11n
        * 802.11g
        * 802.11a
        * 802.11b
    * Legacy 802.11
* Network monitoring and performance tuning
* Network monitoring essentials
* How to use and select access points
* Security
    * security issues unique to wireless networks
    * Wireless security protocols
        * WEP
        * WPA
        * WPA2
        * WPA3.
