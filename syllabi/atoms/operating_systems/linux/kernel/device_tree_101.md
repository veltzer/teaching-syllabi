# Device Tree 101

## Credits
Mark Veltzer [mark.veltzer@gmail.com](mailto:mark.veltzer@gmail.com)

## Description
This short course is about the `Linux` device tree, what is it for and how to manipulate it with the
object of providing board support or tuning your kernel to a specific purpose.

## Duration
8 hours / 1 day

## Outline
* Why the Device Tree ?
* Basic Device Tree syntax
* Device Tree inheritance
* Device Tree specifications and bindings
* Device Tree and `Linux` kernel drivers
* Common properties and examples
