# Introduction to `Cloud Computing`

## Description
A cloud-provider-agnostic introduction to `cloud computing` concepts. This course gives
participants a thorough understanding of what `cloud computing` is, how it works, and how
to evaluate cloud services and providers. It covers the essential terminology, service
models, deployment models, and security considerations that every technology professional
needs to know in today's cloud-first world.

## Duration
16 hours / 1 day

## Intended Audience
* Developers who want to understand cloud infrastructure and services
* System administrators transitioning to cloud environments
* `DevOps` engineers who need a `solid` cloud foundation
* Managers and decision makers evaluating cloud strategies

## Prerequisites
* Basic understanding of computer networking concepts
* Familiarity with operating systems (`Linux` or `Windows`)

## Objectives
* Understand what `cloud computing` is and how it evolved
* Distinguish between the different cloud service models (`IaaS`, `PaaS`, `SaaS`, `FaaS`)
* Understand cloud deployment models and `when` to use each
* Be aware of cloud security fundamentals and the shared responsibility model
* Be able to compare major cloud providers and their offerings
* Understand cloud economics and cost optimization basics
* Know how cloud supports modern `DevOps` practices
