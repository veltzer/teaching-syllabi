# FinOps / Cloud Cost Optimization

## Description
This course provides a comprehensive guide to managing and optimizing cloud costs using FinOps principles and practices. Participants will learn how to implement effective tagging strategies, rightsize resources, leverage reserved instances and savings plans, set up budgets and alerts, utilize spot instances, and apply cost optimization tools. The course also covers organizational best practices for establishing a FinOps culture.

## Duration
8 hours / 1 day

## Intended Audience
Cloud architects, `DevOps` engineers, engineering managers, and finance stakeholders who are responsible for managing or optimizing cloud spending across the organization.

## Prerequisites
* Working knowledge of at least one major cloud provider (`AWS`, `Azure`, or `GCP`)
* Basic understanding of cloud services (compute, storage, networking)
* Familiarity with cloud billing concepts

## Required Knowledge
* Introduction to `Azure` (or equivalent experience)
* `GCP` Fundamentals (or equivalent experience)

## Objectives
* Understand the FinOps framework and its core principles
* Implement effective tagging and cost allocation strategies
* Rightsize cloud resources based on utilization data
* Leverage reserved instances, savings plans, and spot instances to reduce costs
* Set up budgets, alerts, and cost governance policies
* Use cloud-native and third-party cost optimization tools
* Establish organizational best practices for cloud cost management
