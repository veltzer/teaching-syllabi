# Introduction to `AWS`

## Credits
Mark Veltzer [mark.veltzer@gmail.com](mailto:mark.veltzer@gmail.com)

## Description
Cloud Computing is a fundamental shift in how technology services are delivered. As the world's leading cloud platform, `Amazon Web Services` (`AWS`) offers a broad set of global compute, storage, database, analytics, and application services. This course provides a foundational understanding of the `AWS` cloud. Participants will learn about core `AWS` services, key concepts, security, and pricing, equipping them with the knowledge to begin their journey with `AWS`.

We will cover the basic building blocks of the `AWS` platform, focusing on the services and concepts essential for understanding and working within the `AWS` environment.

## Duration
16 hours / 2 days

## Intended Audience
This course is designed for anyone seeking to learn the fundamentals of `Amazon Web Services`. It is suitable for a wide range of attendees, including developers, `IT` professionals, system administrators, project managers, and business leaders who want to understand the value and basics of the `AWS` cloud.

## Prerequisites
General `IT` knowledge and a desire to learn about cloud computing. No prior experience with `AWS` or cloud services is required.

## Objectives
* The participants will be able to understand the core concepts of cloud computing and the `AWS` value proposition.
* They will be able to identify the key services on the `AWS` platform and understand their primary use cases, including compute, storage, networking, and databases.
* Participants will learn the fundamentals of `AWS` security and identity management.
* They will gain familiarity with the `AWS Management Console` and understand the basic principles of `AWS` pricing.
* This course provides a strong foundation for pursuing further `AWS` training and certifications like the `AWS Certified Cloud Practitioner`.

## Outline
* Introduction to Cloud Computing and `AWS`. 2h
    * What is Cloud Computing?
    * Benefits of the `AWS` Cloud
    * `AWS` Global Infrastructure (Regions and Availability Zones)
    * A walkthrough of the `AWS Management Console`
* Core Compute Services. 3h
    * `Amazon EC2` (Elastic Compute Cloud)
    * `Amazon Machine Images` (`AMIs`) and Instance Types
    * Introduction to `Elastic Load Balancing` (`ELB`)
    * Introduction to `Auto Scaling`
* Core Storage Services. 3h
    * `Amazon S3` (Simple Storage Service)
    * `Amazon EBS` (Elastic Block Store)
    * `Amazon EFS` (Elastic File System) vs. `FSx`
    * `Amazon Glacier`
* Networking and Content Delivery. 2h
    * `Amazon VPC` (Virtual Private Cloud)
    * Subnets and Security Groups
    * `Amazon Route 53`
    * `Amazon CloudFront`
* Identity and Access Management. 2h
    * `AWS Identity and Access Management` (`IAM`)
    * Users, Groups, and Roles
    * `IAM` Policies and Best Practices
* Database Services. 2h
    * Database models (`SQL` vs. `NoSQL`)
    * `Amazon RDS` (Relational Database Service)
    * `Amazon DynamoDB`
* Management and Monitoring. 1h
    * `Amazon CloudWatch`
    * `AWS CloudTrail`
* Course Wrap-up. 1h
    * The `AWS Well-Architected Framework`
    * Understanding `AWS` pricing and billing
    * Next Steps and Path to Certification
