# `C17`

## Description
This course covers the `C17` standard (also known as C18), which is an incremental revision of the C language. Participants will learn about the clarifications, defect fixes, and minor improvements introduced in `C17` over C11, and how to adopt these changes in their codebases.

## Duration
8 hours / 1 day

## Intended Audience
C developers who want to stay current with the latest standardized features and improvements in the C language.

## Prerequisites
* `Solid` working knowledge of C programming.
* Familiarity with C11 features is recommended.

## Objectives
* understand the core concepts and principles of `C17`
* gain practical knowledge of Overview of `C17` and its relationship to `C11
* gain practical knowledge of Defect resolutions and clarifications in ```C17``
* gain practical knowledge of Changes to __STDC_VERSION__ and feature test macros
