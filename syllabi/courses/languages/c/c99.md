# C99

## Credits
Mark Veltzer [mark.veltzer@gmail.com](mailto:mark.veltzer@gmail.com)

## Description
A review of the main features of the `C99` standard.

## Duration
8 hours / 1 day

## Intended Audience
`C` developers who want to get acquainted with the `C99` features.

## Prerequisites
3 years of `C` development

## Objectives
* understand the core concepts and principles of C99
* gain practical knowledge of inline` functions
* gain practical knowledge of intermingled declarations and code
* gain practical knowledge of new data types

## Outline
* `inline` functions
* intermingled declarations and code
* new data types
* variable-length `arrays`
* flexible `array` members
* one line comments
* new library functions
* new headers
* type-generic math
* improved `floating point` support
* designated initializers
* compound literals
* variadic `macros`
* `restrict` keyword
* universal character names
* `static` in array indices
