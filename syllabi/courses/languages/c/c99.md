# `C99`

## Description
A review of the main features of the `C99` standard.

## Duration
16 hours / 2 day

## Intended Audience
C developers who want to get acquainted with the `C99` features.

## Prerequisites
3 years of C development

## Objectives
* understand the core concepts and principles of `C99`
* gain practical knowledge of inline` functions
* gain practical knowledge of intermingled declarations and code
* gain practical knowledge of new data types
