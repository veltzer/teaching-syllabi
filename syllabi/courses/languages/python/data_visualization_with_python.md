# Data Visualization with `Python`

## Description
This course covers the essential `Python` libraries and techniques for creating compelling
data visualizations. Participants will learn to build static, statistical, and interactive
charts using `Matplotlib`, `Seaborn`, Plotly, and Dash. The course emphasizes choosing
the right chart type for different data stories, applying color theory and accessibility
principles, creating publication-quality figures, and building interactive web dashboards.
Geospatial visualization techniques are also covered for location-based data analysis.

## Duration
16 hours / 2 days

## Intended Audience
* Developers building data-driven applications with visual components
* Data scientists and data analysts who need to communicate findings through visualizations

## Prerequisites
* Basic understanding of statistics (distributions, correlation, aggregation)

## Required Knowledge
* `Python` Programming (or equivalent experience)

## Objectives
* **Create static visualizations** using `Matplotlib` for publication-quality figures
* **Build statistical plots** using `Seaborn` for exploratory data analysis
* **Develop interactive charts** using Plotly for dynamic data exploration
* **Build web dashboards** with Dash for sharing data insights
* **Apply design principles** including color theory, accessibility, and storytelling with data
* **Visualize geospatial data** using mapping libraries in `Python`
