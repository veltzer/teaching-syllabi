# Introduction to Programming Concepts

## Description
A language-agnostic introduction to programming for absolute beginners entering training programs
or starting their journey in software development. This course covers the fundamental
building blocks of programming that apply across all languages, giving students a `solid`
conceptual foundation before they dive into a specific programming language.

## Duration
16 hours / 1 day

## Intended Audience
* Developers who are just starting their programming journey
* Bootcamp students who need a `solid` foundation before learning a specific language

## Prerequisites
* No prior programming experience required
* Basic computer literacy (using a keyboard, navigating files and folders)

## Objectives
* Understand the fundamental concepts that underpin all programming languages
* Be able to read and reason about simple programs written in pseudo-code
* Understand how data is stored and manipulated using variables and data types
* Know how to control program flow using conditionals and loops
* Understand the purpose and structure of functions
* Be familiar with common data structures and when to use them
* Have a foundation to begin learning any specific programming language
