# `TypeScript` Programming

## Credits
Mark Veltzer [mark.veltzer@gmail.com](mailto:mark.veltzer@gmail.com)

## Description
`TypeScript` is a strongly typed superset of `JavaScript` developed by Microsoft that compiles to plain `JavaScript`. It has become the standard for large-scale `JavaScript` development, providing type safety, better tooling, and improved maintainability. This course covers the `TypeScript` language from the ground up, including its type system, advanced types, and integration with modern development workflows.

## Duration
32 hours / 4 days

## Intended Audience
* `JavaScript` developers who want to add type safety to their code
* Frontend and backend developers building large-scale applications
* Developers new to `TypeScript` who want a comprehensive introduction

## Prerequisites
* Basic `JavaScript` knowledge
* Familiarity with a code editor

## Objectives
* Understand `TypeScript`'s type system and how it builds on `JavaScript`
* Write type-safe code using interfaces, generics, and utility types
* Configure and use the `TypeScript` compiler
* Apply advanced type patterns for real-world applications
* Integrate `TypeScript` into existing `JavaScript` projects
