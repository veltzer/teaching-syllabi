# `PHP` Programming

## Credits
Mark Veltzer [mark.veltzer@gmail.com](mailto:mark.veltzer@gmail.com)

## Description
`PHP` is one of the most widely used server-side programming languages, powering a large portion of the web. Modern `PHP` (8.x) is a significantly improved language with strong typing features, attributes, fibers, and a rich ecosystem. This course covers `PHP` from fundamentals through modern best practices, including object-oriented programming, the composer ecosystem, and integration with web frameworks.

## Duration
32 hours / 4 days

## Intended Audience
* Developers learning `PHP` for web development
* Backend developers who need to maintain or build `PHP` applications
* Developers coming from other languages who want to learn modern `PHP`

## Prerequisites
* Basic programming experience in any language
* Basic understanding of HTML and HTTP

## Objectives
* Write modern `PHP` code using current best practices
* Understand `PHP`'s type system and object-oriented features
* Use Composer for dependency management
* Build web applications with `PHP`
* Interact with databases from `PHP`
