# C++98

## Credits
Mark Veltzer [mark.veltzer@gmail.com](mailto:mark.veltzer@gmail.com)

## Description
This one day course provides an overview of the features introduced by C++98

## Duration
8 hours / 1 day

## Intended Audience
* C++ developers who want to study the C++98 features.

## Prerequisites
* Working with C++ for at least 3 years.

## Objectives
* understand the core concepts and principles of C++98
* gain practical knowledge of Templates
* gain practical knowledge of STL` (Standard Template Library)
* gain practical knowledge of Namespaces

## Outline
* Templates
* `STL` (Standard Template Library)
* Namespaces
* Runtime Type Information (`RTTI`)
* Exception Handling
* Boolean Type
* `Dynamic_cast` Operator
* Mutable Keyword
* Explicit Keyword
* Member Templates
* Operator new/delete
* Covariant Return Types
