# `C++23`

## Description
This one day course provides an overview of the features introduced by `C++23`

## Duration
16 hours / 1 day

## Intended Audience
* `C++` developers who want to study the `C++23` features.

## Prerequisites
* Working with `C++` for at least 3 years.

## Objectives
* understand the core concepts and principles of `C++23`
* gain practical knowledge of if `consteval`
* gain practical knowledge of `size_t` `Range-based for-Loops`
* gain practical knowledge of Deducing this
