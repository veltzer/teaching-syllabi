# C++14

## Credits
Mark Veltzer [mark.veltzer@gmail.com](mailto:mark.veltzer@gmail.com)

## Description
This one day course provides an overview of the features introduced by C++14

## Duration
8 hours / 1 day

## Intended Audience
* C++ developers who want to study the C++14 features.

## Prerequisites
* Working with C++ for at least 3 years.

## Outline
* Generic Lambdas
* Return Type Deduction
* Variable Templates
* Relaxed `constexpr`
* Binary Literals
* Digit Separators
* Deprecated Attribute
* Shared Timed Mutex
* Make_unique
* Extended Capturing in Lambdas
