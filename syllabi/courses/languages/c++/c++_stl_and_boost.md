# `C++` `STL` and `Boost`

## Description
This course various topics related to `STL` and `Boost` which are commonly used in the industry and their pitfalls.

## Duration
24 hours / 3 days

## Intended Audience
* `C++` developers
* software engineers working on performance-critical applications

## Prerequisites
* experience with `C++` programming

## Objectives
* understand the core concepts and principles of `C++` `STL` and ```Boost``
* gain practical knowledge of `STL`
* gain practical knowledge of `Boost`
* gain practical knowledge of Exercises

## Outline
<!-- chapter: stl, duration: 7h -->
* `STL`
    * Performance of various `STL` data structures.
    * Correct iteration of data structures.
    * Modifying containers while iterating.
    * `STL` allocators.
    * Complex keys for maps, sets...
<!-- chapter: boost, duration: 11h -->
* `Boost`
    * Bind.
    * Flyweight.
    * Tuple.
    * Circular buffer.
    * Intrusive containers (compare to `STL`).
    * Pool.
    * Scope Exit.
    * Serialization.
    * Tokenizer (compare to `STL`, `ACE`).
<!-- chapter: exercises, duration: 6h -->
* Exercises
    * read and write records from the disk using the serialization (`boost`).
    * use the pool for allocation.
    * use Flyweight to hold the objects.

## Notes
* Each topic will be expounded by examples
