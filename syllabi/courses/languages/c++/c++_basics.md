# `C++` Basics

## Credits
Mark Veltzer [mark.veltzer@gmail.com](mailto:mark.veltzer@gmail.com)

## Description
This is a basic course for `C++` developers introducing the `C++` language to them and going over the basic
concepts of the language.

## Duration
24 hours / 3 days

## Intended Audience
* C++ developers
* software engineers working on performance-critical applications

## Prerequisites
* experience with C++ programming

## Objectives
* understand the core concepts and principles of C++` Basics
* gain practical knowledge of Introduction
* gain practical knowledge of What are objects?
* gain practical knowledge of Classes, Constructors, Destructors

## Outline
* Introduction
* What are objects?
* Classes, Constructors, Destructors
* Operators `new` & `delete`
* Reference
* `Const` Specifier
* Overloading and Mangled Name
* Self Reference - `this`
* Assignment Operator and Copy Constructor
* Initialization list
* Standard `C++` IO
* `Static` mem-functions and data-members
* `Inline` Specifier
* Operator Overloading
* Inheritance
* Polymorphism
* `ABC` - Abstract Base Class
* Friends
* Exception Handler
* `Templates`
* `RTTI`
* `C++` File and String Streams
* Combining `C` and `C++`
* `STL` (Standard Template Library - brief) 339
    * Introduction
    * Containers
    * Iterators
    * Sequence containers
    * Predicates
    * Associative containers
    * Algorithms
    * How to choose `STL` Container
    * Safe `STL`
