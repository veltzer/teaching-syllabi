# `Android` Application Development

## Description
`Android` is the world's most widely used mobile operating system, powering billions of devices globally. This course provides a comprehensive guide to building modern `Android` applications using `Kotlin` and `Android Studio`. Students will learn to design responsive user interfaces, manage application lifecycle, persist data, communicate over the network, and follow modern architectural patterns including MVVM and `Jetpack Compose`.

## Duration
32 hours / 4 days

## Intended Audience
* Developers who want to build native `Android` applications
* Developers transitioning from web or backend development to mobile
* Developers looking to modernize their `Android` skills with `Jetpack Compose`

## Prerequisites
* Programming experience in `Java` or `Kotlin`
* Basic understanding of object-oriented programming

## Required Knowledge
* `Kotlin` Programming (or equivalent experience)

## Objectives
* Set up and use `Android Studio` effectively
* Build user interfaces with `XML` layouts and `Jetpack Compose`
* Manage activities, fragments, and the application lifecycle
* Persist data using Room and SharedPreferences
* Communicate with `REST` APIs using Retrofit
* Apply the MVVM architectural pattern
* Handle permissions and prepare apps for publishing
