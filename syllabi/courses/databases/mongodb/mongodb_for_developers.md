# `MongoDB` for Developers

## Credits
Mark Veltzer [mark.veltzer@gmail.com](mailto:mark.veltzer@gmail.com)

## Description
`MongoDB` is a leading `NoSQL` document database that enables developers to build modern, scalable applications with flexible data models. This course provides comprehensive training for developers who need to design, build, and optimize applications using `MongoDB`. Participants will learn modern development patterns, performance optimization techniques, and best practices for building production-ready applications.

The course covers both fundamental concepts for newcomers to `NoSQL` and advanced techniques for experienced developers, focusing on practical application development rather than database administration.

This course targets `MongoDB` version 7 and includes coverage of `MongoDB Atlas` cloud platform.

## Duration
24 hours / 3 days

## Intended Audience
* Application developers building `MongoDB`-backed applications
* Backend developers working with `NoSQL` databases
* Full-stack developers needing database skills
* Software engineers migrating from relational databases
* `API` developers building data-driven services
* Developers with or without prior database experience

## Prerequisites
* Proficiency in at least one programming language (`JavaScript`, `Python`, `Java`, or `C#` preferred)
* Basic understanding of `JSON` data format
* Familiarity with `REST APIs` and web development concepts
* Basic command line skills

## Objectives
* Design effective data models for `MongoDB` applications
* Implement efficient queries using modern query patterns
* Build performant applications using aggregation pipelines
* Implement transactions for data consistency
* Optimize application performance through proper indexing
* Integrate `MongoDB` with modern application frameworks
* Apply security best practices in application development
* Handle real-time data with change streams

## Exercises
Hands-on coding exercises using popular `MongoDB` drivers in the student's preferred programming language. Students will build a complete application from scratch, implementing features like user authentication, data validation, complex queries, aggregation pipelines, transactions, and real-time updates. Exercises include working with `MongoDB Atlas`, implementing proper error handling, optimizing query performance, and integrating with modern web frameworks. Real-world scenarios cover e-commerce, social media, and `IoT` application patterns.

## Installations
Each student should have:

* Development machine with 8 GB RAM minimum
* Choice of operating system (`Windows`, `macOS`, or `Linux`)
* `MongoDB` Community Edition 7.x or `MongoDB Atlas` free tier account
* `MongoDB Compass` GUI tool (latest version)
* Modern `IDE` or code editor (`VS Code` recommended with `MongoDB` extension)
* Programming language environment:
    * `Node.js` 18+ with `npm` for `JavaScript` developers
    * `Python` 3.9+ with `pip` for `Python` developers
    * `JDK` 11+ with `Maven`/`Gradle` for `Java` developers
    * `.NET` 6+ `SDK` for `C#` developers
* `Git` for version control
* `Docker Desktop` (optional but recommended)
* `Postman` or similar tool for `API` testing
* Free internet access for package installation and `MongoDB Atlas` access
* Sample datasets will be provided during the course
