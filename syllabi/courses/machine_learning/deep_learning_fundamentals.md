# `Deep Learning` Fundamentals

## Description
This course covers the foundations and practical application of `deep learning`. Participants will learn
how neural networks work, and gain hands-on experience building and training models using `PyTorch`
and `TensorFlow`. The course progresses from basic feedforward networks through convolutional and
recurrent architectures to modern topics including transformers, transfer learning, and generative
adversarial networks.

## Duration
24 hours / 3 days

## Intended Audience
* Developers with `machine learning` experience who want to dive into `deep learning`.
* Data scientists looking to leverage neural network architectures for complex problems.
* Engineers who want to understand and apply modern `deep learning` techniques.

## Prerequisites
* `Solid` `Python` programming skills.
* Basic understanding of `machine learning` concepts (supervised/unsupervised learning, model evaluation).
* Familiarity with `numpy` and linear algebra fundamentals.

## Required Knowledge
* `Python` Programming (or equivalent experience)

## Objectives
* Understand the theory behind neural networks and backpropagation.
* Build and train `deep learning` models using `PyTorch` and `TensorFlow`.
* Apply CNNs to image-related tasks.
* Use RNNs and LSTMs for sequential data processing.
* Understand the transformer architecture and its applications.
* Leverage transfer learning to accelerate model development.
* Understand the principles behind GANs and generative models.

* Neural Network Foundations

    * Activation functions
