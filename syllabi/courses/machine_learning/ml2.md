# Machine learning

## Credits
Mark Veltzer [mark.veltzer@gmail.com](mailto:mark.veltzer@gmail.com)

## Description

## Duration
56 hours / 7 days

## Intended Audience
* data scientists and machine learning engineers
* software developers transitioning to ML roles

## Prerequisites
* programming experience in Python
* basic knowledge of statistics and linear algebra

## Objectives
* understand the core concepts and principles of Machine learning
* gain practical knowledge of Basics and supervised learning
* gain practical knowledge of Unsupervised learning and data handling
* gain practical knowledge of Deep learning basics

## Outline
* Basics and supervised learning
    * Intro to `ML`
    * Linear Regression as an example
    * Gradient descent- theory
    * Gradient descent- methods
    * Supervised learning theory and formalization
    * Hypothesis classes
    * Over-fitting and regularization
    * Regularized linear regression
    * Logistic regression
    * Multi-Class Logistic regression
    * Other optimization techniques
* Unsupervised learning and data handling
    * Intro to unsupervised learning, Data science examples
    * Dimensionality reduction: PCA (Principal component analysis)
    * K nearest neighbors
    * Fast algorithms for nearest neighbors
    * What is clustering?
    * K-means
    * Clustering algorithms: Hierarchical, DBSCAN, OPTICS
    * Handling big data - introduction
* Deep learning basics
    * Intro and history
    * Neural networks
    * Properties of a neural net
    * Back propagation
    * Gradient descent reminder
    * Gradient descent in deep learning
    * Back propagation
    * Why does it work?
    * Tips and tricks
* Advanced deep learning
    * Convolutional neural nets: intro
    * Convolutional neural nets: architecture, tips examples
    * Auto Encoders
    * RNN (Recurrent Neural Nets)
    * LSTM (Long Short Term Memory)
    * Generative Models
    * (GAN) Generative Adversarial Networks
* More topics and algorithms
    * SVM - Support Vector Machines
    * Random forests
    * Kernel trick
    * Statistical learning
    * Markov models and algorithms
    * Introduction to Reinforcement Learning
    * Deep Reinforcement learning
* Applications and recent advances
    * Introduction to `NLP` (Natural Language Processing)
    * Introduction to Computer vision
    * Methods and examples of deep learning for `NLP` and CV
    * Recent papers and active research
    * Tips and tricks
