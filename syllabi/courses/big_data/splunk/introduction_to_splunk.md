# Introduction to `Splunk`

## Credits
Mark Veltzer [mark.veltzer@gmail.com](mailto:mark.veltzer@gmail.com)

## Description
The Introduction to `Splunk` training course provides an overview of `Splunk`, an operational intelligence, log management, application management, enterprise security and compliance providing application.
The course begins with a lesson on how to use `Splunk` to run basic searches and the fundamentals of `Splunk`. Next, the course dives into stats, tags, and even types. The course concludes with a lesson on creating reports.

## Duration
16 hours / 2 days

## Intended Audience
* `DevOps` engineers with 3-6 months experience

## Prerequisites
* 3-6 months experience in `DevOps`

## Objectives
* Utilize `Splunk` to run basic searches.
* Identify the fundamentals of a `Splunk` search.
* Discover the stats command and its functions.
* Create reports.

## Outline
* Start Searching
    * `Splunk` architecture overview (indexers, search heads, forwarders)
    * Introduce `Splunk` and the Search app
    * Run basic searches
    * Time range selection and time-based searching
    * What `Splunk` is not good for
    * Recommended Log Formats
    * Identify the contents of search results
    * Control a search job
    * Use the output of a search to refine your search
* Fundamentals
    * Examine the anatomy of a search
    * Understand search language syntax concepts
    * Review fields and use the fields command
    * Create a table
    * Examine multivalue fields
* Stats
    * Understand the stats command
    * Display top and rare values for given fields
    * Use the stats command to create statistical reports
* Data Transformation
    * Understand the `eval` command
    * Perform calculations on field values
    * Convert, round, and format field values
    * Use conditional statements
    * Using REX
    * Tags and event types
* Creating Reports and Dashboards
    * Create reports and charts
    * Chart functions (timechart, sparkline)
    * Split values into multiple series
    * Omit null and other values from charts
    * Apply statistical functions
    * Introduction to dashboards
    * Alerts and scheduled searches
