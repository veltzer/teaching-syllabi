# Object-Oriented Programming

## Description
This course provides a thorough foundation in object-oriented programming principles and practices. Participants will learn to think in terms of objects and classes, understand the four pillars of `OOP`, and apply design principles that lead to maintainable and extensible software. The course uses examples in both `Python` and `Java` to illustrate concepts across different language paradigms.

The course bridges theory and practice, covering everything from basic class design through `SOLID` principles and `UML` class diagrams, equipping participants to write well-structured object-oriented code.

## Duration
16 hours / 2 days

## Intended Audience
* Software developers seeking a `solid` `OOP` foundation
* Developers transitioning from procedural to object-oriented programming
* Testers who need to understand object-oriented codebases
* Programmers preparing for `design patterns` study

## Prerequisites
* Proficiency in at least one programming language
* Basic understanding of functions, variables, and control flow
* Familiarity with either `Python` or `Java` is helpful but not required

## Required Knowledge
* `Python` Programming (or equivalent experience)

## Objectives
* Understand the core principles of object-oriented programming
* Design classes with proper encapsulation and clear interfaces
* Apply inheritance and polymorphism effectively
* Distinguish between composition and inheritance and choose appropriately
* Apply `SOLID` principles to produce maintainable designs
* Read and create `UML` class diagrams
* Recognize and avoid common `OOP` pitfalls
