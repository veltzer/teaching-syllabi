# Introduction to Game Development with `Unity`

## Description
This course will provide a comprehensive introduction to game development using the `Unity` game engine. Students will learn the core concepts of game design, 3D modeling, scripting, animation, and user interface design, culminating in the creation of their own interactive projects.

## Duration
24 hours / 3 days

## Intended Audience
* aspiring game developers
* software developers interested in game development

## Prerequisites
* Basic computer literacy
* No prior programming or game development experience required

## Objectives
* Navigate the `Unity` interface and understand its key components
* Utilize 3D modeling tools to create and import game assets
* Write `C#` scripts to control game logic and behavior
* Implement animations to bring characters and objects to life
* Design user interfaces for player interaction
* Apply game design principles to create engaging experiences
* Build and deploy their own `Unity` projects
