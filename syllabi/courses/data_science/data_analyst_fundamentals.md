# Data Analyst Fundamentals

## Description
This course provides a comprehensive foundation for aspiring and junior data analysts. Participants will
learn the complete data analysis workflow from data acquisition and cleaning through visualization and
reporting, using industry-standard tools including `SQL`, `Python`, Excel, and Tableau.

## Duration
24 hours / 3 days

## Intended Audience
* Aspiring data analysts looking to build foundational skills
* Developers transitioning into data analysis roles
* Managers who want to understand data analysis processes and outputs

## Prerequisites
* Basic computer literacy
* Familiarity with spreadsheets
* Logical thinking and basic math skills

## Objectives
* Understand the end-to-end data analysis workflow
* Write `SQL` queries to extract and manipulate data
* Clean and prepare data for analysis
* Perform exploratory data analysis and apply statistical fundamentals
* Create effective visualizations using `Matplotlib`, `Seaborn`, and Tableau
* Build dashboards and reports that communicate insights clearly
* Tell compelling stories with data
