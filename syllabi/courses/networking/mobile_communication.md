# Mobile Communication

## Description
This course provides a comprehensive overview of mobile communication technologies, from foundational cellular concepts through the latest `5G NR` architecture. Participants will learn the evolution of cellular networks across generations, understand radio access technologies, and explore the architecture of modern mobile systems. The course covers network planning fundamentals, spectrum management, and emerging topics such as network slicing and IoT connectivity.

The course is designed for professionals who need a `solid` understanding of mobile communication systems for development, architecture, or integration work.

## Duration
16 hours / 2 days

## Intended Audience
* Software developers building mobile and telecom applications
* System architects designing solutions for cellular networks
* Embedded engineers working with cellular modules and IoT devices
* Engineers transitioning into the telecom domain

## Prerequisites
* Basic understanding of networking concepts (`IP`, `TCP`/`UDP`)
* Familiarity with radio frequency concepts is helpful but not required
* General understanding of computer architecture

## Objectives
* Understand cellular network architecture and evolution
* Explain the key technologies in each cellular generation (2G through 5G)
* Describe `4G LTE` architecture including EPC and eNodeB
* Understand `5G NR` architecture, network slicing, and mmWave
* Analyze radio access technologies and spectrum management
* Explain handover mechanisms and mobility management
* Evaluate IoT connectivity options (NB-IoT, LTE-M)
