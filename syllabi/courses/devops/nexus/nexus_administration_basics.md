# `Nexus` Administration Basics

## Credits
Mark Veltzer [mark.veltzer@gmail.com](mailto:mark.veltzer@gmail.com)

## Description
This course is designed to provide new customers with the first steps towards optimizing their `Nexus` Repository Manager configuration. Specifically, it provides critical, high-level theory, best practice, and practical application related to understanding specific concepts and terminology related to `Nexus` Repository Manager.

## Duration
4 hours / 0.5 days

## Intended Audience
* Users interested in learning the first steps towards optimizing `Nexus` Repository Manager

## Prerequisites
* Knowledge about Software Development Life-cycle (SDLC)
* Knowledge about modern `DevOps` practices
* Knowledge about continuous integration and delivery
* Knowledge about continuous integration and delivery

## Objectives
* Learn the basic concepts of repository managers
* Learn the basic concepts of `Nexus` as a repository manager
* Learn the best practices of using `Nexus`

## Outline
* Functions of a Repository Manager
    * Functions of a Repository Manager
    * Repository vs. Repository Manager: What's the Difference?
    * Repo for SDLC Management
* Components and Formats
* Repository Types
* Repository Planning Checklist
