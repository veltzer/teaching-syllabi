Description

This course is a practical course and is intended for people just entering the IT industry with no hands on experience with how testing and integration is done in the real world.

At first the course introduces the participants to the ideas of CI/CD and git as one of the most used tools for source code management

Then the course introduces the participants to web architecture basics and to HTML, CSS and JavaScript.

The main part of the course comes next. After the first two introductory chapters the course delves into Python and Selenium as a testing framework for professional web testing.

Duration

70 hours / 9 days

Intended Audience

  • IT Beginners who wish to better understand CI/CD and IT workflow and Selenium in particular.

Prerequisites

  • Tech affinity.
  • Programming in some language is an advantage.
  • Knowledge in web fundamentals is an advantage.
  • Python is an advantage.

Objectives

  • Understand basic CI/CD concepts
  • Have a basic understanding of how git works
  • Be able to use git basic commands
  • Be able to describe the basics of the web architecture.
  • Be able to write HTML, CSS, and JavaScript code at a basic level.
  • Understand the basic concepts behind Selenium.
  • Be able to write tests for web applications using Selenium.
