Using Bash

Credits

Mark Veltzer mark.veltzer@gmail.com

Description

Bash is the de-facto shell for Linux and by extension of the modern cloud. This course teaches how to use that shell effectively and explains topics such as environment variables, configuration files, filesystem layout and many more.

Duration

16 hours / 2 days
