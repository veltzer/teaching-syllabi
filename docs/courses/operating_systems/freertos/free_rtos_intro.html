Description

FreeRTOS is fast becoming an industry standard as an operating system for Real Time systems on micro controllers and small microprocessors. This course introduces developers to FreeRTOS and teaches how write effective software for this OS kernel.

Duration

80 hours / 10 days

Intended Audience

  • Developers who wish to learn the features of FreeRTOS
  • Developers who want to write efficient code for FreeRTOS

Prerequisites

  • C programming (C++ an advantage)
  • Knowledge of other operating systems (Linux, Windows, …) is an advantage.
  • Knowledge of other real time systems (Linux, …) is an advantage.

Objectives

  • understand the core concepts and principles of FreeRTOS` intro
  • gain practical knowledge of Demo projects
  • gain practical knowledge of Coding standard
  • gain practical knowledge of Tasks

Outline

  • Demo projects
  • Coding standard
  • Tasks
  • Co-routines
  • Queues
  • Binary Semaphores
  • Counting Semaphores
  • Mutexes
  • Recursive Mutexes
  • Task Notifications
  • Software Timers
  • Creating a new project
  • Memory management
  • Stack overflow protection
  • Hook functions
