Description

The debugging course aims to give programmers and team leaders a broader view that debugging plays in systems engineering. In addition this course introduces the participants to tools they did not know existed. This combination enables to build software which is easier to debug both at the development and client sites. This ultimately saves precious time and produces systems which are more stable and more extendable and delays the stasis that most C/C++ systems reach at some point.

Duration

24 hours / 3 days

Prerequisites

  • The course is ONLY for C/C++ developers.
  • The course is ONLY for UNIX/Linux developers.
  • The course is ONLY for programmers who have been programming for at least one year.
