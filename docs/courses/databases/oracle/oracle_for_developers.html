Oracle for Developers

Credits

Description

This course is designed to provide students with a comprehensive understanding of Oracle Database and its development environment. Students will learn how to design, implement, and manage databases using Oracle’s tools and technologies. The course covers various aspects of database development, including SQL, PL/SQL, database administration, and performance tuning.

Duration

40 hours / 5 days

Intended Audience

  • software developers working with databases
  • backend engineers and data engineers

Prerequisites

  • Basic knowledge of programming concepts
  • Familiarity with relational database management systems (RDBMS)

Objectives

Upon successful completion of this course, students will be able to: * Understand the architecture and components of Oracle Database * Design and create database objects (tables, views, indexes, etc.) * Write SQL queries for data manipulation and retrieval * Develop stored procedures, functions, and triggers using PL/SQL * Implement database security and user management * Perform database backup and recovery operations * Optimize database performance using indexing and other techniques * Integrate Oracle Database with various development environments and tools

Outline

  • Introduction to Oracle Database
    • Overview of Oracle Database
    • Database architecture and components
    • Oracle software installation and configuration
  • SQL Fundamentals
    • Data Definition Language (DDL) and Data Manipulation Language (DML)
    • Creating and modifying database objects (tables, views, indexes)
    • Querying data using SQL (SELECT, JOIN, subqueries, etc.)
    • Data manipulation (INSERT, UPDATE, DELETE)
    • SQL functions and operators
  • PL/SQL Programming
    • Introduction to PL/SQL
    • PL/SQL block structure and syntax
    • Variables and data types
    • Control structures (loops, conditionals)
    • Stored procedures and functions
    • Triggers and event handling
    • Error handling and exception management
    • Cursors and cursor manipulation
  • Database Administration
    • User and role management
    • Database security (privileges, roles, virtual private databases)
    • Backup and recovery strategies
    • Import and export utilities
    • Performance monitoring and tuning
  • Advanced SQL
    • Analytic functions and window functions
    • Hierarchical queries and recursive SQL
    • Regular expressions and pattern matching
    • Data warehousing and OLAP concepts
  • Database Performance Tuning
    • Indexing strategies
    • Query optimization techniques
    • Monitoring and analyzing database performance
    • Memory management and configuration
  • Oracle Development Tools
    • SQL Developer
    • Data Modeler
    • SQL*Plus and other utilities
    • Integrating Oracle with programming languages (Java, Python, etc.)
  • Database Integration and Application Development
    • Connecting to Oracle from various development environments
    • Oracle Call Interface (OCI) and JDBC
    • Building database* driven applications
    • Best practices for application development

Textbooks and Resources

  • “Oracle PL/SQL Programming” by Steven Feuerstein
  • “Oracle Database 12c: The Complete Reference” by Bob Bryla and Kevin Loney
  • Online resources (Oracle Learning Library, Oracle Developer Community, etc.)
