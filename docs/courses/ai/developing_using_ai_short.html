Description

This one-day intensive course covers the essentials of AI-assisted software development. Participants will learn to effectively use AI coding assistants, apply prompt engineering techniques for code generation and debugging, and integrate AI tools into their daily development workflows. The course focuses on practical, hands-on experience with the most impactful AI development techniques that can be immediately applied to real work.

Duration

8 hours / 1 day

Intended Audience

  • Professional software developers seeking to integrate AI tools into their workflow
  • Development team leads evaluating AI adoption strategies
  • DevOps engineers interested in AI-powered automation

Prerequisites

  • Proficiency in at least one programming language (Python, JavaScript, Java, or similar)
  • Experience with software development practices (version control, testing, debugging)
  • Familiarity with development environments and IDEs

Objectives

  • Configure and effectively utilize AI coding assistants in daily development workflows
  • Apply prompt engineering techniques to generate high-quality code and solve programming problems
  • Implement AI-enhanced development practices including testing, refactoring, and debugging
  • Understand the limitations, security considerations, and best practices for AI-assisted development
