C Programming

Description

C is and has been for years the world’s best language for low level, systems and embedded programming. This course covers this language and teaches how to program in it.

Duration

32 hours / 4 days

Intended Audience

  • C programmers
  • embedded and systems software developers

Prerequisites

  • basic programming experience in any language

Objectives

  • understand the core concepts and principles of C Programming
  • gain practical knowledge of Introduction to C
  • gain practical knowledge of Basic Syntax and Data Types
  • gain practical knowledge of Control Flow

Outline

  • Introduction to C
    • History and importance of C
    • C language standards (ANSI C, C99, C11, C17)
    • Setting up the development environment
    • Structure of a C program
    • Compilation process
    • First program: Hello World
  • Basic Syntax and Data Types
    • Variables and constants
    • Basic data types (int, float, double, char)
    • Type modifiers (short, long, signed, unsigned)
    • Type conversion and casting
    • Operators (arithmetic, relational, logical, bitwise)
    • Expressions and statements
  • Control Flow
    • Conditional statements (if, if-else, switch)
    • Loops (while, do-while, for)
    • Break and continue statements
    • Goto and labels
    • Logical expressions and short-circuit evaluation
  • Functions
    • Function declaration and definition
    • Function parameters and return values
    • Call by value vs. call by reference
    • Scope and lifetime of variables
    • Storage classes (auto, static, extern, register)
    • Recursive functions
  • Arrays and Strings
    • One-dimensional arrays
    • Multi-dimensional arrays
    • Array initialization
    • Character arrays and strings
    • String handling functions
    • String library (string.h)
  • Pointers
    • Pointer fundamentals
    • Pointer arithmetic
    • Pointers and arrays
    • Pointers and functions
    • Multiple indirection (pointers to pointers)
    • Void pointers
    • Function pointers
  • Memory Management
    • Memory models and layout
    • Dynamic memory allocation (malloc, calloc, realloc, free)
    • Memory leaks and dangling pointers
    • Stack vs. heap memory
  • Structures and Unions
    • Defining and using structures
    • Nested structures
    • Arrays of structures
    • Structures and functions
    • Structure pointers
    • Unions and their applications
    • Bit-fields
    • Typedef
  • File I/O
    • File handling concepts
    • Text and binary files
    • File operations (open, close, read, write)
    • Error handling in file operations
    • Random access to files
    • Command line arguments
  • Module 10: Preprocessor Directives
    • #include directive
    • Macro definition (#define)
    • Conditional compilation (#if, #ifdef, etc.)
    • Predefined macros
    • #pragma directives
  • Advanced Topics
    • Enumerations
    • Complex declarations
    • Variable arguments functions
    • Bit manipulation
    • Signals and error handling
    • Multi-file programming
    • Header files and their organization
    • Creating and using libraries
  • Standard Library
    • Standard I/O (stdio.h)
    • Character handling (ctype.h)
    • Mathematics functions (math.h)
    • Utility functions (stdlib.h)
    • Time and date functions (time.h)
  • Best Practices and Design
    • Writing efficient C code
    • Debugging techniques
    • Code organization and style
    • Documentation practices
    • Common C programming pitfalls
