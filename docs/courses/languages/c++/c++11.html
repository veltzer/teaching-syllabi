C++11

Credits

Mark Veltzer mark.veltzer@gmail.com

Description

This one day course provides an overview of the features introduced by C++11

Duration

8 hours / 1 day

Intended Audience

  • C++ developers who want to study the C++11 features.

Prerequisites

  • Working with C++ for at least 3 years.

Outline

  • Lambda Expressions
  • Auto Type Deduction
  • Nullptr
  • Range-based for Loop
  • Uniform Initialization
  • Rvalue References & Move Semantics
  • Strongly-typed Enums
  • Smart Pointers (unique_ptr, shared_ptr)
  • Threading Support (thread, mutex, future)
  • Variadic Templates
  • Static Assertions
  • Delegating & Inherited Constructors
  • Override & Final Keywords
  • Default & Delete Functions
  • Constexpr
