C++23

Credits

Mark Veltzer mark.veltzer@gmail.com

Description

This one day course provides an overview of the features introduced by C++23

Duration

8 hours / 1 day

Intended Audience

  • C++ developers who want to study the C++23 features.

Prerequisites

  • Working with C++ for at least 3 years.

Objectives

  • understand the core concepts and principles of C++23
  • gain practical knowledge of if consteval
  • gain practical knowledge of size_t` Range-based for Loops
  • gain practical knowledge of Deducing this

Outline

  • if consteval
  • size_t Range-based for Loops
  • Deducing this
  • Multidimensional Subscript Operator
  • Static Operator[]
  • std::expected
  • std::flat_map/set
  • std::mdspan
  • Lambda Support in unevaluated contexts
  • std::print
  • std::move_only_function
  • Modules (improvements)
  • Unicode Support
  • std::ranges Improvements
