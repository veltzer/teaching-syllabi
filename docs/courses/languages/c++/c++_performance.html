C++ Performance

Credits

Mark Veltzer mark.veltzer@gmail.com

Description

This course deals with how to write C++ code with emphasis on performance. We deal with the cost of various C++ features and with interacting efficiently with the operating system.

Duration

16 hours / 2 days

Outline

  • Micro optimizations
  • C++ efficiency compared to C and assembly
  • Is efficiency important?
  • What type of efficiency?
  • The fast path
  • Basics (how classes, inheritance, virtuals, etc are implemented)
  • Inlining
  • Efficiency topics
