Description

A full and comprehensive course about the Java programming language and environment.

Duration

  • 2 weeks for experienced programmers
  • 3 weeks for non experienced programmers with more emphasis on exercises

Intended Audience

This course is intended for developers who wish to study the basics of the Java programming language

Prerequisites
