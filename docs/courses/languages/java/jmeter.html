Description

This course covers Apache JMeter, the open-source load testing tool, from basic test creation through advanced capabilities. Students will learn to record, build, and run load tests, analyze results, handle high loads with distributed testing, and test HTTP, FTP, and database endpoints.

Duration

16 hours / 2 days

Intended Audience

  • Java developers
  • software engineers working with JVM-based technologies

Prerequisites

  • experience with Java programming

Objectives

  • understand the core concepts and principles of JMeter
  • gain practical knowledge of High level intro to load testing
  • gain practical knowledge of JMeter` basics
  • gain practical knowledge of Real test cases

Outline

  • High level intro to load testing
  • JMeter basics
    • Installing
    • The UI
    • Manual test creation - JMeter basic test elements
    • Running a test
    • Analyzing results using several listeners
  • Real test cases
    • Recording real web events
    • Advanced test elements (waits, loops, managers, …)
    • Using data from external CSV file (usernames, password, …)
  • Advanced capabilities
    • JMeter advanced test elements - (Controllers, listeners, …)
    • Extracting values from responses into variables
    • Asserts
    • Plugins
    • Ramp-up and Ramp-down
    • Set up and tear down
    • Debugging tests
  • Mobile recording
    • Android
    • iOS
  • Handling high loads
    • Running tests from the CLI (Non GUI mode)
    • Using load slaves (Distributed testing)
    • Handling common issues
  • Misc
    • Working With HTTP Requests
    • Working With FTP Request
    • Database Testing With JMeter
    • JMeter Variables And Functions
    • Distributed (Remote) Testing
    • Practice time - hands on time to do some real tests with the JMeter on our Application.
