Intermediate Splunk

Credits

Mark Veltzer mark.veltzer@gmail.com

Description

This course is intended for Splunk users who would like to understand Splunk better. It does with some of the more advanced features of Splunk and is intended for users who use Splunk regularly.

Duration

16 hours / 2 days

Intended Audience

  • People working with Splunk for up to one year and wish to understand Splunk more deeply.

Prerequisites

  • People working with Splunk for up to one year.

Objectives

  • Brush up on Splunk search basics
  • Learn more advanced search techniques
  • Get to know Splunk visualizations
  • Understand how to create and use fields
  • Learn how to use tag and events
  • Learn about data models.

Outline

  • Course Introduction
  • Beyond Search Fundamentals
  • Commands for Visualizations
  • Advanced Visualizations
  • Filtering and Formatting Data
  • Correlating Events
  • Introduction to Knowledge Objects
  • Creating and Managing Fields
  • Creating Field Aliases and Calculated Fields
  • Creating Tags and Event Types
  • Creating and Using Macros
  • Creating and Using Workflow Actions
  • Creating Data Models
  • Using the Common Information Model (CIM) Add-On
